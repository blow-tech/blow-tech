My work focuses on PowerShell, Bash, Active Directory, Windows Server, Linux administration, and operational scripting for real-world IT environments.

- Active Directory, DNS, DHCP, GPO, and account lifecycle management

- Microsoft 365 and Exchange Online administration
- Git and technical documentation

## Featured Projects

### [powershell-admin-toolkit](https://github.com/blow-tech/powershell-admin-toolkit)

A PowerShell toolkit for Windows Server, Active Directory, Exchange Online, Microsoft 365, and infrastructure administration.

Includes scripts for:

- Active Directory reporting
- user logon and lockout investigation
- password expiry notifications
- system inventory
- backup and cleanup tasks
- administrative automation

### [linux-admin-scripts](https://github.com/blow-tech/linux-admin-scripts)

A Bash scripting repository for Linux system administration, server auditing, monitoring, and health checks.

Includes scripts for:

- server health checks
- disk and memory monitoring
- log review
- backup validation
- service status reporting
- routine maintenance tasks
