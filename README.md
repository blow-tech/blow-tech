## blow-tech

Systems administrator and infrastructure engineer. Enterprise AD, M365, and Linux
environments — building automation around what used to be manual work.

---

## Repositories

### [powershell-admin-toolkit](https://github.com/blow-tech/powershell-admin-toolkit)

Production PowerShell toolkit for Windows Server, Active Directory, Exchange Online,
and M365 administration. Built from real incidents — not tutorials. Covers AD lifecycle
reporting, Exchange auditing, environment health checks, certificate expiry sweeps, and
disk reclamation, with HTML report output throughout.

`PowerShell 5.1+` `Active Directory` `Exchange Online` `Microsoft 365` `PKI / Certificates`

---

### [linux-admin-scripts](https://github.com/blow-tech/linux-admin-scripts)

Bash scripting repository for RHEL server administration, auditing, monitoring, and
maintenance. Covers health checks with threshold alerting, rotated/compressed log
parsing, systemd service monitoring, backup validation, and routine cleanup — structured
for scheduled execution and log output.

`Bash` `RHEL / CentOS` `systemd` `Log Parsing` `Monitoring`
