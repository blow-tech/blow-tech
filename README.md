## 🚀 Featured Projects

### 🛠️ [powershell-admin-toolkit](https://github.com/blow-tech/powershell-admin-toolkit)
> A production-ready PowerShell toolkit for Windows Server, Active Directory, Exchange Online, and Microsoft 365 administration.

Scripts built around real operational tasks:

- **AD reporting** — expiring accounts, last logon, lockout source tracing, logon/logoff audit
- **Exchange Online** — mailbox permissions, DL membership, shared mailbox sizing, external mail tracing
- **Environment health** — AD replication, DNS, disk, services, certificates, event logs in one HTML report
- **Backup & cleanup** — AD state backup, automated disk reclamation
- **Inventory** — full system info snapshots, environment audits

---

### 🐧 [linux-admin-scripts](https://github.com/blow-tech/linux-admin-scripts)
> A Bash scripting repository for Linux system administration, server auditing, monitoring, and health checks.

Scripts covering daily Linux ops:

- **Health checks** — CPU, memory, disk, load average with threshold alerting
- **Log review** — rotated/compressed log parsing, error pattern extraction
- **Service monitoring** — systemd service status reporting and restart handling
- **Backup validation** — verify backup integrity and file presence
- **Routine maintenance** — scheduled cleanup, temp file management, package audit
